Controls programm for Mantigora HV device

Functionality

Top menu allow call next widget: list of devices connected to PC, log of program, inforamation about device parameters and this help. Any device open in separate tab. Device tab separate on two area: be left locate controls element and by right locate oscilloscope. Recorder --- allow turn on record indicator's values to file (in append mode) in format: unixtimestap, voltage(V), currect(mcA) Indicator --- show voltage and current on device with fixed period. Oscilloscope --- show graph for voltage and current for last 10 minutes. Allow take screenshots, stop oscliscope, save osciliscope buffer and customize graph style and views
